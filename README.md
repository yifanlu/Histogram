Histogram
=========

This is a simple command line tool that generates a histogram of byte distribution of a given file.
This is useful in data analysis, for example an extremely even byte distribution usually indicates encryption, or a large amount of 0xE1 may indicate ARM binary, and so on.
The tool is designed to be very small and portable, and should compile on any POSIX system with nothing more than stdlib.

Building
---------

I did not include a Makefile because that would take me more time than writing the tool itself. Compiling should be as easy as:
```
gcc -o histogram histogram.c
```
